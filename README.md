# ProblemK-cpp
Решение задачи ProblemK при помощи C++ ( https://wiki.haskell.org/Ru/Problem_K )

В состоянии разработки

# Компиляция
```
(> C:/ProblemK-cpp)
mkdir build
cd build
cmake ..
cd ..
cmake --build build
```
